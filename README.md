\# Projet Shoot Them Up

\*\*Auteur :\*\* Ezechiel



Description

Ce projet est un jeu de type Shoot Them Up développé en Python avec PyGame dans le cadre du module "Outils de gestion".



Fonctionnalités prévues:

\- Un vaisseau spatial contrôlé par le joueur.

\- Des ennemis qui descendent vers le bas de l'écran.
